## Critical Issues (Errors)

### 1. Structured Data - Invalid JSON-LD (100 pages)

All pages have schema validation errors:

- Missing `@context`
- `Product.offers.price` is required
- `Product.offers.availability` is required

### 2. Accessibility - Form Labels Missing (44 pages)

Form inputs without labels on tool pages and `/blog`

### 3. Images - Oversized Files (7 pages)

Two images on `/tools/smart-ai-enhancement` are huge:

- `budget-edit-with-smart-AI.png` = **1.9 MB**
- `budget-edit-with-WITHOUT-smart-AI.png` = **1.6 MB**

### 4. Content - Meta Tags in Body (1 page)

`/blog` has 6 meta tags incorrectly placed in `<body>`

### 5. Core SEO - Missing H1 (1 page)

Homepage `/` has no H1 tag

### 6. Security - Leaked Secrets (1 high-confidence)

Google OAuth Client ID exposed in JS bundle
