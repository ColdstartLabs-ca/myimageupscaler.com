## Implementation Notes

1. **Title tag rules**:
   - Max 60 characters (ideal: 50-60)
   - Front-load the keyword
   - Include year if content is dated (2026)

2. **H1 rules**:
   - Should match or closely mirror title tag
   - Include target keyword naturally
   - Avoid clickbait

3. **First paragraph rules**:
   - Keyword in first sentence
   - Clear value proposition
   - Match search intent

4. **After making changes**:
   - Request indexing in GSC (URL Inspection tool)
   - Wait 7-14 days for position changes
   - Don't change more than 3-5 pages at once

## The Three Kings Checklist

For each optimized page, verify:

- [ ] **King 1 (Title)**: Keyword present, front-loaded, under 60 chars
- [ ] **King 2 (H1)**: Keyword present, natural phrasing
- [ ] **King 3 (First Paragraph)**: Keyword in first sentence
- [ ] **Meta Description**: Compelling, under 155 chars, includes CTA
- [ ] **URL**: Short, descriptive, includes keyword if possible
- [ ] **Internal Links**: 2-3 related internal links
- [ ] **External Links**: 1-2 authoritative sources (if applicable)
